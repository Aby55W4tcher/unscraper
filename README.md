# unscraper
This is a web scraper built to extract pdf's from selected
parts of the following pages:

UNOCHA.org
UNHCR.org
UNICEF.org
WEP.org
UNEP.org
UNODC.org
OPCW.org
WTO.org
UNISDR.org
UNAIDS.org
IAEA.org
FAO.org
ILO.org
WHO.int

and then create a seperate csv file for each of these with a list of
the pdf's.
This program is set to run every 24hrs.
